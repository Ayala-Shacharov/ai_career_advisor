# AI Career Recommendation System

## 📌 Overview
This project is an AI-powered career recommendation system.  
The system collects user input and generates personalized questions, then analyzes answers to suggest the most suitable profession.

## ⚙️ How it works

1. The user enters a free-text description about themselves.
2. The system uses AI to generate 4 personalized multiple-choice questions.
3. The user answers the questions.
4. The system analyzes the responses and returns the most suitable career recommendation.

## 🧠 Core Idea
Instead of using fixed logic for all users, the system dynamically adapts both the questions and the final recommendation using AI, making the experience personalized for each user.

## 🚀 Features
- AI-generated personalized questions
- Interactive questionnaire (multiple choice)
- Career recommendation based on answers
- Dynamic logic instead of static rules

## 🛠️ Tech Stack
- React (Frontend)
- Node.js / Express (Backend)
- AI API integration

## 📁 Project Structure
Frontend handles UI and user interaction.  
Backend handles AI requests and recommendation logic.

## 🎯 Goal
To demonstrate how AI can be used to personalize decision-making systems and improve user experience.
